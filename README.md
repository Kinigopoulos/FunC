# FunC Project

#### You can use the following commands:
# `npm run dev`
Starts <b>both</b> front-end and back-end code<br/>
Front: http://localhost:3000/<br/>
Back: http://localhost:5000/

### `npm start`
Starts only front-end code on<br/>
http://localhost:3000/

### `npm run server`
Starts only back-end code on<br/>
http://localhost:5000/

### `npm run build`
Builds front-end
