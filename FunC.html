FunC is a coding website for beginners. Our goal is to make programming fun
and exciting for people who are getting into computer science and coding.
Here you can navigate a selection of tutorials that will teach you the basics of
programming on Java, C++ and also the query Language SQL. Each programming
language has 20+ tutorials each one with multiple tests that provide you with a
chance to write your own code.
